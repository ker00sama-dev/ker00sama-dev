## Featured work — KooAds

An ad monetization platform (offerwall) connecting publishers and advertisers, built for a large company. I was the primary developer across the API, admin panel, CMS, documentation, and public website.

Laravel backend, headless Next.js 16 frontend (React 19, Tailwind CSS 4, ISR with on-demand revalidation), PostgreSQL, Meilisearch, and a complete SEO layer — structured data, sitemaps, and an `llms.txt` route for AI crawlers.
